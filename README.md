## HOW TO USE THIS APP ?

1. Clone the project

```bash
git clone <url of the repo>
```

2. Install the dependencies for the front and the back

```bash
npm install
```

3. Start the application in the **frontend** folder

```bash
npm start
```

## Docker

```bash
- docker-compose up -d --build

```
